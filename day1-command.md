# Day1 Linux 基础命令（文件/目录操作）
## 核心命令
| 命令 | 作用 | 示例 |
|------|------|------|
| pwd | 查看当前路径 | pwd |
| ls/ll | 列出目录内容 | ll -a（显示隐藏文件） |
| cd | 切换目录 | cd ~（回家目录）、cd ..（上一级） |
| mkdir | 创建目录 | mkdir -p a/b/c |
| touch | 创建空文件 | touch test.txt |
| cp | 复制 | cp -r 源目录 目标目录 |
| mv | 移动/重命名 | mv old.txt new.txt |
| rm | 删除 | rm -rf 目录（谨慎使用） |
