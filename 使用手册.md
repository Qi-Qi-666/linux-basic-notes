# 数据集小工具 使用手册
## 1. 启动工具
打开WSL终端，输入：
cd ~/linux-basic-notes
uvicorn fastapi_sql_day4:app --reload --host 0.0.0.0

## 2. 查WSL IP
新开WSL终端，输入：hostname -I

## 3. 常用功能
- 查数据统计：浏览器输 http://WSLIP:8000/api/label_stats
- 上传CSV数据：浏览器输 http://WSLIP:8000/docs → 点/api/upload_csv → Try it out → 选文件上传
- 查单条数据：浏览器输 http://WSLIP:8000/api/get_data/数据ID（比如1/2/3）
